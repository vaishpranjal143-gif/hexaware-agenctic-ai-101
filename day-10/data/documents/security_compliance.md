# Caramel Assistant — Security & Compliance

Caramel AI takes data security seriously and maintains the following
certifications and practices:

**Certifications.** Caramel AI is SOC 2 Type II certified and ISO 27001
certified. The Enterprise plan additionally supports HIPAA-compliant
deployments for healthcare customers via a signed Business Associate
Agreement (BAA).

**Data Encryption.** All customer data is encrypted at rest using AES-256 and
in transit using TLS 1.3. Encryption keys are rotated every 90 days.

**Data Residency.** Enterprise customers can choose to host their data in
the US, EU, or India region to meet local data residency requirements. By
default, data is stored in the US region.

**Data Retention.** Conversation logs are retained for 24 months by default
and can be configured down to 30 days for customers with stricter retention
policies. Customers can request full data deletion at any time via the
admin dashboard.

**Access Control.** Caramel AI supports SSO via SAML 2.0 and OAuth 2.0, along
with role-based access control (RBAC) with four roles: Viewer, Agent, Admin,
and Owner.

**Sub-processors.** Caramel AI uses Microsoft Azure for compute and storage
infrastructure, and a list of all sub-processors is published and updated on
the company's trust portal.
